Scanning ports from range

Ports with status OPEN are 100% open. Some applications that I have detected with 100% accuracy so far: discord, battle.net.

Ports with status Likely open are potentially open, the algorithm can easily be improved to be more accurate, a lot of false positives are currently included.

Initial testing performed on Windows 11 + Chrome 110
